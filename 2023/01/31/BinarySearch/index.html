二分查找の注意事项

  • 循环不变量的确定 == 区间的确定

循环不变量：每次循环中保持不变的性质

区间定义一般分为两种：

  • 左闭右开
  • 左闭右闭

左闭右开

形如 $[left, right)$ 的形式，此时 $left == right$ 毫无意义，因此控制循环的条件应当为(left < right)，即结束循环的条件为 $left == right$ ，而关于左右边界的确定为：

  • 若 $ target < mid $ ，说明目标数存在于左半边，因此下一次搜索区间应当为 $[left, mid - 1]$ ，即 $[left, mid)$ ，由于存在循环不变量，即同原始区间 $[left, right)$ 保持相同的格式（$right$所指向位置不被包括在循环以内），因此此时 $right = mid$ 。
  • 若 $target > mid$ ，说明目标数存在于右半边，因此下一次搜索区间应当为 $[mid + 1, right)$ ，同上，存在循环不变量，因此此时 $left = mid + 1$。

左闭右闭

形如 $[left, right]$ 的形式，此时 $left == right$ 有意义，因此控制循环的条件应当为(left <= right)，即结束循环的条件为 $left == right + 1$ ，而关于左右边界的确定为：

  • 若 $ target < mid $ ，说明目标数存在于左半边，因此下一次搜索区间应当为 $[left, mid - 1]$ ，由于存在循环不变量，即同原始区间 $[left, right]$ 保持相同的格式（$right$所指向位置包括在循环以内），因此此时 $right = mid - 1$ 。
  • 若 $target > mid$ ，说明目标数存在于右半边，因此下一次搜索区间应当为 $[mid + 1, right]$ ，同上，存在循环不变量，因此此时 $left = mid + 1$。

Q：既然第一种情况下的区间为 $[left, mid)$ 那为什么不直接将 $right$ 赋值为 $mid - 1$？

A：考虑一个含有两个元素的有序数组 $[1, 2]$ 目标查找数为 $1$ 采用左闭右开的形式，此时 $left = 0$ ，$right = 2$ ，$ mid = 1$ 。此时 $1 < nums[mid] = 2$ 倘若 $right$ 按照 $mid - 1$ 来赋值，那么下一次循环区间变为 $[0, 0]$ 而循环终止条件为 $left == right$ ，此时注意到 $1$ 并未得到遍历就结束了循环。造成这种现象的原因应该是，问题的这种行为导致了循环不变量（搜索区间）的改变，导致有一部分数并未进行新一轮搜索而被忽略终止了。
