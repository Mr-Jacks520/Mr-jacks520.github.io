二叉树的层序遍历

给你一个二叉树，请你返回其按 层序遍历 得到的节点值。 （即逐层地，从左到右访问所有节点）。

本题二叉树的层序遍历需要利用队列进行层层遍历，利用这种模板就可以解决大部分的层序遍历题，这种层序遍历也就相当于图论中的广度优先遍历。

当然，还是需要理解深度优先遍历的思想。

二叉树的层序遍历II

给定一个二叉树，返回其节点值自底向上的层次遍历。 （即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

  • 本题只要在层序遍历的基础上，将结果进行翻转即可。

  • 但是在Java中我们仍然可以利用链表的头插法将插入时间复杂度降低，很妙啊。

二叉树的右视图

给定一棵二叉树，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

解法一中，思路是先一次性遍历完右子树，之后再遍历左子树。

解法二中，思路为采用双端队列+BFS思想，每次将每一层的最右结点加入到答案即可。

二叉树的层平均值

给定一个非空二叉树的根节点 root , 以数组的形式返回每一层节点的平均值。与实际答案相差 10-5 以内的答案可以被接受。

本题没啥好说的，直接经典层序遍历后求和求平均即可。

填充每个结点的下一个右侧节点指针

给定一个 完美二叉树 ，其所有叶子节点都在同一层，每个父节点都有两个子节点。二叉树定义如下：

struct Node {
int val;
Node *left;
Node *right;
Node *next;
}
填充它的每个 next 指针，让这个指针指向其下一个右侧节点。如果找不到下一个右侧节点，则将 next 指针设置为 NULL。

初始状态下，所有 next 指针都被设置为 NULL。

层序遍历连接即可。

利用递归也是个不错的解决方法。

二叉树的最大深度

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：
给定二叉树 [3,9,20,null,null,15,7]，

    3

/
9 20
/
15 7
返回它的最大深度 3 。

最小深度是从根节点到最近叶子节点的最短路径上的节点数量。
