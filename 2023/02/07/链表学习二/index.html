  • 哈系表做法利用一个集合保存其中一个链表遍历后的结点，然后再去遍历另一个链表，查到了就是有，没查到就是没有。
  • 差值做法目的是想把两个指针的起点搞成一致，然后一起往前走，碰到了就是有，没碰到就没有
  • 数学做法就更加巧妙了，基于数学证明，最终两个指针肯定能碰到一起。

数学证明：

设链表A不相交部分长度为a，链表B不相交部分长度为b，相交部分长度为c。则在两个指针分别经过 a + c + b 和 b + c + a 的路程之后就会相遇在相交点！
